# Мобильное приложение для управления окнами

Это маленькое PWA-приложение для Android и iPhone. Оно управляет ESP32-контроллером через уже существующие API:

- `GET /api/window?target=...`
- `POST /window/cmd`
- `GET /api/rs485`

## Самый простой способ

После прошивки ESP32 откройте на телефоне:

```text
http://IP_ESP32/mobile
```

Например:

```text
http://192.168.100.5/mobile
```

Введите логин и пароль от веб-интерфейса ESP32. После открытия страницу можно добавить на главный экран:

- Android Chrome: меню `...` -> `Добавить на главный экран`
- iPhone Safari: `Поделиться` -> `На экран Домой`

## Что есть в приложении

- выбор окна/RP2040;
- кнопки `Открыто`, `Закрыто`, `Проветривание`, `Стоп`;
- настройки только для соединения с ESP32: адрес, логин, пароль;
- автоматическая загрузка локальных RP2040 и включенных RS-485 узлов.

## Отдельный PWA-вариант

Файлы `index.html`, `manifest.json` и `sw.js` можно разместить на любом HTTP-сервере в локальной сети. Для iPhone удобнее использовать встроенную страницу ESP32 `/mobile`, потому что она не упирается в ограничения HTTPS/HTTP.
